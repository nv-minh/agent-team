# EM-Backend-Expert - Backend Specialist

Backend development expert specializing in APIs, databases, and performance optimization.

## Usage

```
Use the em-backend-expert skill to [task description]
```

## Examples

```
Use the em-backend-expert skill to review the API performance
Use the em-backend-expert skill to optimize the database queries
Use the em-backend-expert skill to design the REST API architecture
Use the em-backend-expert skill to troubleshoot the database connection issue
```

## What This Does

This is a shortcut that invokes the **em-backend-expert agent** to:
- Review API design and performance
- Optimize database queries and schema
- Design scalable backend architecture
- Troubleshoot backend issues
- Implement backend best practices

The em-backend-expert agent specializes in:
