# EM:Bug-Fix - Bug Fixing

Systematic workflow for investigating and fixing bugs.

## Usage

```
Use the em:bug-fix skill to fix [bug description]
```

## Examples

```
Use the em:bug-fix skill to fix the login timeout issue
Use the em:bug-fix skill to fix the database connection error
Use the em:bug-fix skill to fix the memory leak
Use the em:bug-fix skill to fix the API response format issue
```

## What This Does

Invokes the **em-bug-fix workflow** to:
- Investigate the bug systematically
- Identify root cause
- Create fix plan
- Implement the fix
- Test the fix
- Verify resolution
