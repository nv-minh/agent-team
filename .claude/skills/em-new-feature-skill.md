# EM-New-Feature - Feature Implementation Workflow

Complete workflow for implementing features from idea to production.

## Usage

```
Use the em-new-feature skill to [feature description]
```

## Examples

```
Use the em-new-feature skill to implement user authentication with JWT
Use the em-new-feature skill to build a shopping cart feature
Use the em-new-feature skill to create a user profile system
Use the em-new-feature skill to add email notifications
```

## What This Does

This is a shortcut that invokes the **em-new-feature workflow** to:
- Create feature specification
- Design architecture
- Implement feature
- Write tests
- Review code
- Deploy feature

## Workflow: EM-New-Feature

The em-new-feature workflow covers:
1. **Define**: Requirements and specifications
2. **Plan**: Implementation strategy
3. **Build**: Code implementation
4. **Verify**: Testing and validation
5. **Review**: Code review and QA
6. **Simplify**: Refactor and optimize
7. **Ship**: Deploy and monitor

Trigger this with: `Use the em-new-feature skill to [feature]`
