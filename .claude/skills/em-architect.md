# EM:Architect - Architecture Design

Design and review software architecture.

## Usage

```
Use the em:architect skill to [design/review] [system/component]
```

## Examples

```
Use the em:architect skill to design the microservices architecture
Use the em:architect skill to review the system architecture
Use the em:architect skill to design the API gateway architecture
Use the em:architect skill to evaluate the current architecture
```

## What This Does

Invokes the **em-architect agent** to:
- Design system architecture
- Review architectural decisions
- Suggest architectural patterns
- Evaluate trade-offs
- Plan technical roadmap
