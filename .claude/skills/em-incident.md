# EM:Incident - Incident Response

Handle production incidents systematically.

## Usage

```
Use the em:incident skill to handle [incident description]
```

## Examples

```
Use the em:incident skill to handle the database connection failure
Use the em:incident skill to handle the API latency spike
Use the em:incident skill to handle the memory leak in production
Use the em:incident skill to handle the payment processing error
```

## What This Does

Invokes the **em-incident-response workflow** to:
- Assess incident severity
- Coordinate response team
- Investigate root cause
- Implement mitigation
- Verify resolution
- Document postmortem
- Prevent recurrence
