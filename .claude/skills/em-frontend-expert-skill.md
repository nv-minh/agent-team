# EM-Frontend-Expert - Frontend Specialist

Frontend development expert specializing in React, Next.js, UI/UX, and performance.

## Usage

```
Use the em-frontend-expert skill to [task description]
```

## Examples

```
Use the em-frontend-expert skill to review the React components
Use the em-frontend-expert skill to optimize the Next.js app performance
Use the em-frontend-expert skill to design the user profile page
Use the em-frontend-expert skill to refactor the component architecture
```

## What This Does

This is a shortcut that invokes the **em-frontend-expert agent** to:
- Review React/Next.js code
- Optimize frontend performance
- Design UI/UX components
- Implement frontend best practices
- Refactor component architecture
