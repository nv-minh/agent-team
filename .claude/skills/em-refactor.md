# EM:Refactor - Code Refactoring

Improve code quality through systematic refactoring.

## Usage

```
Use the em:refactor skill to refactor [code/component]
```

## Examples

```
Use the em:refactor skill to refactor the authentication module
Use the em:refactor skill to refactor the API handlers
Use the em:refactor skill to refactor the React components
Use the em:refactor skill to refactor the database queries
```

## What This Does

Invokes the **em-refactoring workflow** to:
- Analyze code quality issues
- Identify refactoring opportunities
- Apply refactoring patterns
- Improve code structure
- Maintain functionality
- Ensure tests pass
