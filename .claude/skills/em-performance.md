# EM:Performance - Performance Optimization

Benchmark and optimize system performance.

## Usage

```
Use the em:performance skill to [optimize/benchmark] [component/system]
```

## Examples

```
Use the em:performance skill to optimize the API response time
Use the em:performance skill to benchmark the database queries
Use the em:performance skill to optimize the frontend bundle size
Use the em:performance skill to analyze the application performance
```

## What This Does

Invokes the **em-performance-auditor agent** to:
- Benchmark system performance
- Identify bottlenecks
- Suggest optimizations
- Apply performance patterns
- Measure improvements
- Ensure scalability
