# EM:Code-Review - Code Review

Perform 5-axis code review for quality and correctness.

## Usage

```
Use the em:code-review skill to review [code/PR/changes]
```

## Examples

```
Use the em:code-review skill to review the authentication module
Use the em:code-review skill to review PR #123
Use the em:code-review skill to review the recent changes
Use the em:code-review skill to review the API endpoints
```

## What This Does

Invokes the **em-code-reviewer agent** to:
- Perform 5-axis code review
- Check code quality
- Identify potential bugs
- Suggest improvements
- Ensure best practices
