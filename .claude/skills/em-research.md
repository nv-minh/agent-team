# EM:Research - Technical Research

Research and explore technical topics and solutions.

## Usage

```
Use the em:research skill to research [topic/technology]
```

## Examples

```
Use the em:research skill to research the best authentication libraries
Use the em:research skill to explore GraphQL vs REST API
Use the em:research skill to research database options for the project
Use the em:research skill to explore frontend frameworks for the new feature
```

## What This Does

Invokes the **em-researcher agent** to:
- Research technical topics
- Compare solutions and technologies
- Evaluate trade-offs
- Provide recommendations
- Document findings
