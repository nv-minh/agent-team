# EM:Security - Security Audit

Perform security audits using OWASP Top 10 guidelines.

## Usage

```
Use the em:security skill to audit [component/system]
```

## Examples

```
Use the em:security skill to audit the authentication system
Use the em:security skill to audit the payment processing
Use the em:security skill to audit the API endpoints
Use the em:security skill to audit the user input handling
```

## What This Does

Invokes the **em-security-auditor agent** to:
- Perform OWASP Top 10 assessment
- Identify security vulnerabilities
- Suggest security improvements
- Apply security best practices
- Ensure compliance
