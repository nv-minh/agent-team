# EM:Test - Test Engineering

Test strategy, auto-generation from source code, video recording for E2E evidence, and API contract testing with timing validation.

## Usage

```
Use the em:test skill to [task description]
```

## Examples

```
Use the em:test skill to create a test strategy for the authentication module
Use the em:test skill to generate unit tests for the API endpoints
Use the em:test skill to set up video recording for E2E tests
Use the em:test skill to benchmark API response times
Use the em:test skill to design integration tests for the payment system
```

## What This Does

Invokes the **em-test-engineer agent** to:
- Create comprehensive test strategies
- Auto-generate test cases from source code analysis
- Generate unit, integration, and E2E tests
- Configure video recording and evidence collection
- Validate API contracts with response time benchmarks
- Review test coverage
- Ensure quality standards
