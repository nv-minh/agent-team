# EM:Planner - Implementation Planning

Creates detailed implementation plans for features and tasks.

## Usage

```
Use the em:planner skill to [task description]
```

## Examples

```
Use the em:planner skill to create an implementation plan for JWT authentication
Use the em:planner skill to plan the database migration for user profiles
Use the em:planner skill to design the REST API for the shopping cart
```

## What This Does

Invokes the **em-planner agent** to:
- Break down tasks into steps
- Identify dependencies
- Estimate effort
- Suggest implementation approach
- Create verification criteria
