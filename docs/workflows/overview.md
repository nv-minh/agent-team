# Workflows Overview

Complete catalog of EM-Skill workflows.

## Primary Workflows
1. **new-feature** - From idea to production
2. **bug-fix** - Investigate and fix bugs
3. **refactoring** - Improve code quality
4. **security-audit** - Security assessment

## Support Workflows
5. **project-setup** - Initialize new projects
6. **documentation** - Generate and update docs
7. **deployment** - Deploy and monitor
8. **retro** - Learn and improve

## Distributed Workflows
9. **distributed-investigation** - Parallel bug investigation
10. **distributed-development** - Parallel feature implementation

## Team Workflows (11-18)
11. **team-review** - Full team review
12. **architecture-review** - Architecture review
13. **design-review** - UI/UX review
14. **code-review-9axis** - Deep 9-axis review
15. **database-review** - Database review
16. **product-review** - Product review
17. **security-review-advanced** - Advanced security
18. **incident-response** - Incident handling

See [Usage Guide](../guides/usage-guide.md#using-workflows) for details.
